Esta página lista todos los elementos HTML. Están agrupados por funciones para ayudarte a encontrar lo que tienes en mente con facilidad. Aunque esta guía está escrita para aquellos que son nuevos escribiendo código, se pretende que sea una referencia útil para cualquiera.

Elementos BásicosSection

Los elementos básicos son la espina dorsal de cualquier documento HTML. Verás estos elementos en el código fuente en todas las páginas web, después de la declaración del tipo de documento « doctype » en la primera línea de la página. El « doctype » especifica qué versión de (X)HTML esta usando la página. Los elementos que conforman el contenido de una página web se colocan entre la etiqueta de apertura <html> y la etiqueta de cierre </html>. El elemento <html> también se conoce como elemento raíz.

Element Description
<html>

Metadatos del documentoSection

Los metadatos contienen información sobre la página. Esto incluye información sobre estilos, scripts y datos que ayudan al software (search engines, browsers, etc.) a usar y generar la página. Los metadatos de estilos y scripts pueden estar definidos en la página o estar enlazados a otro fichero que contiene la información.

Element Description
<link>
<meta>
<style>
<title>

Componentes WebSection

Los Componentes Web son una tecnología relacionada con HTML que hacen que sea posible, en esencia, crear y personalizar elementos como si fueran HTML normal. Además, pueden crear versiones personalizadas de los elementos HTML estándar.

Nota: Los elementos de los Componentes Web están definidos en World Wide Web Consortium (W3C) Web Components collection of specifications en lugar de la especificación HTML. Además, la especificación de los Componentes Web no ha sido finalizado y está sujeto a cambios.

Element Description
<content>
<element>
<shadow>
<slot>
<template>
